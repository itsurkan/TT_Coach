# 🏓 AI Coach - Інструкції для запуску

## ✅ Завершено: Базова навігація

Створено 4 основні екрани додатку:

### 📱 Структура навігації

```
WelcomeActivity (Головний екран - LAUNCHER)
    ├── ExerciseSelectionActivity (Вибір вправи)
    │   └── TrainingActivity (Тренування)
    └── SettingsActivity (Налаштування)
```

### 📂 Створені файли

#### Kotlin Activity files:
- `WelcomeActivity.kt` - Головний екран з кнопками "Почати тренування" та "Налаштування"
- `ExerciseSelectionActivity.kt` - Список вправ для вибору
- `ExerciseAdapter.kt` - Адаптер для RecyclerView списку вправ
- `TrainingActivity.kt` - Екран тренування з камерою та фідбеком
- `SettingsActivity.kt` - Налаштування параметрів вправи та додатку

#### Layout XML files:
- `activity_welcome.xml` - UI головного екрану
- `activity_exercise_selection.xml` - UI екрану вибору вправи
- `item_exercise.xml` - Card layout для окремої вправи
- `activity_training.xml` - UI екрану тренування
- `activity_settings.xml` - UI екрану налаштувань

#### Resources:
- `arrays.xml` - Масиви для Spinner (роздільна здатність камери)
- `AndroidManifest.xml` - Оновлено з усіма Activity

### 🚀 Як запустити

1. **Відкрийте проект в Android Studio**
   ```bash
   cd d:\Desktop\TT_Coach_AI
   # Відкрийте папку в Android Studio
   ```

2. **Синхронізуйте Gradle**
   - Android Studio автоматично запропонує синхронізацію
   - Або: File → Sync Project with Gradle Files

3. **Підключіть Android пристрій або емулятор**
   - Мінімальний SDK: 24 (Android 7.0)
   - Рекомендовано: SDK 34 (Android 14)

4. **Запустіть додаток**
   - Натисніть кнопку Run (▶) або
   - `Shift + F10`

### 📋 Функціонал кожного екрану

#### 1️⃣ WelcomeActivity (Головний екран)
- Заголовок "AI Coach - Настільний теніс"
- Кнопка "Почати тренування" → перехід до ExerciseSelectionActivity
- Кнопка "Налаштування" → перехід до SettingsActivity
- Кнопка "Про додаток" → діалог з інформацією
- Підтвердження виходу при натисканні Back

#### 2️⃣ ExerciseSelectionActivity (Вибір вправи)
- RecyclerView зі списком вправ:
  - ✅ "Накат справа" (доступна)
  - 🔒 "Накат зліва" (заблокована)
  - 🔒 "Топ-спін справа" (заблокована)
  - 🔒 "Подача" (заблокована)
- При виборі доступної вправи → перехід до TrainingActivity
- При виборі заблокованої → діалог "Буде доступна у наступних версіях"

#### 3️⃣ TrainingActivity (Тренування)
- Інформація про вправу
- Кнопка "Калібрувати позицію" (симуляція калібрування 3 сек)
- Кнопка "Почати/Зупинити тренування"
- Кнопка "Відкрити камеру" → інтеграція з існуючим CameraActivity
- Поточний фідбек (великий текст з кольором)
- Історія останніх 5 фідбеків
- Симуляція аналізу з рандомним фідбеком:
  - ✅ "Чудово! Ідеальна техніка!"
  - ⚠️ "Більше ротації корпусу!"
  - ⚠️ "Тримай зап'ястя рівно!"
  - і т.д.
- Підсумок тренування при зупинці

#### 4️⃣ SettingsActivity (Налаштування)
**Параметри вправи "Накат справа":**
- Ідеальний кут зап'ястя (0-200°, default: 180°)
- Мінімальна ротація корпусу (0-90°, default: 45°)
- Кут проведення (0-180°, default: 120°)

**Налаштування аудіо:**
- Увімкнути/вимкнути аудіо фідбек
- Гучність (0-100%, default: 80%)
- Швидкість мовлення (0.0-2.0x, default: 1.0x)

**Налаштування камери:**
- Роздільна здатність (480p / 720p / 1080p)
- FPS обробки (15-60, default: 30)
- Показувати скелет на відео

- Кнопка "Зберегти налаштування" → збереження в SharedPreferences
- Кнопка "Скинути до стандартних"
