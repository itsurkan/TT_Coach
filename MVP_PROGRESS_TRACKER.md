## 📝 Нотатки та Рішення

### Технічні рішення:
- **Платформа:** Flutter (крос-платформний)
- **CV фреймворк:** MediaPipe Pose (on-device)
- **TTS:** flutter_tts (native)
- **Сховище:** SharedPreferences (локальне)
- **Опціонально:** OpenAI API для складного фідбеку

### Архітектурні рішення:
```
lib/
├── main.dart
├── screens/
│   ├── home_screen.dart
│   ├── exercise_selection_screen.dart
│   ├── training_screen.dart
│   └── settings_screen.dart
├── services/
│   ├── mediapipe_service.dart
│   ├── motion_analyzer.dart
│   ├── feedback_generator.dart
│   └── tts_service.dart
├── models/
│   ├── exercise_parameters.dart
│   ├── pose_data.dart
│   └── analysis_result.dart
└── utils/
    ├── angle_calculator.dart
    └── motion_detector.dart
```

### Розширення для майбутніх версій:
- [ ] Аналіз м'яча (YOLO + Kalman)

- [ ] Gamification (досягнення, челенджі)
